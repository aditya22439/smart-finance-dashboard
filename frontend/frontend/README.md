# Smart Finance Dashboard

Smart Finance Dashboard is a modern MERN finance application for expense tracking, budget management, and analytics.

## Available Scripts

In the project directory, you can run:

### `npm start`

Runs the frontend in development mode.

### `npm test`

Launches the test runner in interactive watch mode.

### `npm run build`

Builds the frontend for production in the `build` folder.

The production build is optimized and ready for deployment.
